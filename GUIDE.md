# Wafer Map Analyzer - 분석기 가이드

## ATE 테스트 데이터 시각화 개요
ATE 테스트 결과는 각 Die별 전기적 측정값으로 구성되며, Spec 비교 결과에 따라 Pass/Fail 또는 Bin 번호로 분류됩니다.

본 UI는 CSV 업로드 기능을 통해 해당 데이터를 불러오고, 각 Die를 X-Y 좌표로 매핑하여 Wafer Map 형태로 시각화합니다.

## 주요 기능
- **Fail Map 시각화**: Bin 값에 따라 색상을 구분하여 Fail Map을 구성합니다.
- **Die 상세 분석**: 특정 Die 클릭 시 상세 전기 특성을 확인할 수 있습니다.
- **공정 이상 분석**: Yield 계산 및 Center/Edge Fail 분석 기능을 통해 공정 이상 패턴을 빠르게 파악할 수 있도록 설계되었습니다.

## 사용 방법
1. **데이터 업로드**: `CSV 업로드` 기능을 통해 측정 데이터 파일을 불러옵니다.
2. **웨이퍼 탐색**: 마우스 휠로 확대/축소하고 드래그하여 이동합니다.
3. **Spec 설정**: 오른쪽 패널에서 Spec Limit을 조절하여 위반 사항을 확인합니다.
4. **통계 확인**: 상단 바에서 실시간 Yield 및 불량 패턴 분석 결과를 모니터링합니다.
